Historia do Html

A história do HTML remonta ao início da década de 1990, quando foi criado pelo físico Tim Berners-Lee para estruturar documentos científicos e facilitar o compartilhamento de informações. Desde então, a linguagem de marcação evoluiu e se tornou o alicerce para a criação da World Wide Web. Breve história do HTML - HTML e CSS Pro História do HTML: Da criação à evolução da web | PG Tech A História Do HTML O que você precisa saber antes de aprender HTML? - balta.io HTML - conceito, história, como funciona e tags O criador O HTML (HyperText Markup Language) foi desenvolvido em 1991 por Tim Berners-Lee, um físico que trabalhava no CERN, o laboratório europeu de física de partículas na Suíça. A ideia original era criar um sistema de hipertexto para que pesquisadores pudessem compartilhar e acessar documentos de maneira fácil e interconectada. O projeto, junto com o protocolo HTTP e o primeiro navegador, foi o marco para o surgimento da web como a conhecemos hoje. A evolução das versões A evolução do HTML foi marcada pelo lançamento de novas versões, que adicionaram funcionalidades e padronizaram o seu uso na internet. HTML 1.0 (1993): A primeira versão pública tinha recursos básicos, com apenas 18 tags para a criação de documentos com texto e links. HTML 2.0 (1995): Padronizada pelo IETF (Internet Engineering Task Force), essa versão incluiu suporte a formulários e marcações que ainda são usadas atualmente. HTML 3.2 (1997): Lançado pelo World Wide Web Consortium (W3C), trouxe inovações importantes como tabelas e a possibilidade de organizar layouts mais complexos. HTML 4.01 (1999): Estabeleceu a separação entre a estrutura do conteúdo (HTML) e a apresentação visual (CSS), incentivando o uso de folhas de estilo. XHTML (2000): Baseado em XML, o XHTML aplicava regras mais rígidas de sintaxe, visando uma maior compatibilidade e processamento por máquinas. Embora tenha sido considerado o sucessor do HTML por um tempo, não se tornou o padrão dominante.
